# 04-Web-APIs--Code-Quiz
Web application where the user can solve a code quiz. The Following quiz provides Multiple Choice Questions (MCQs) related to Javascript Framework. You will have to read all the given answers and click over the correct answer. 
 
## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

```
$ git clone git@github.com:Cherjios/04-Web-APIs--Code-Quiz.git
$ cd 04-Web-APIs--Code-Quiz
```

Then open in your preferred text editor:
- [vim](https://www.vim.org/) 
- [emmacs](https://www.gnu.org/software/emacs/)
- [visual studio code](https://code.visualstudio.com/)
